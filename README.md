## Getting Started

### Requirements

- [Node.js](https://nodejs.org/download/release/latest-fermium/)
- [yarn](https://yarnpkg.com/)

<br>

### Instalation

Install dependecies with `yarn`

```bash
yarn install
```

<br>

### Development Execution

```bash
yarn dev
```

<br>

### Lint Execution

```bash
yarn lint
```

<br>

### Production Execution

```bash
yarn build
yarn start
```
